Configuration

Alice uses environment variables for global config and settings.json per instance.

Environment Variables

VariableDescription
ALICE_DEFAULT_API_KEYLLM API key (shared across instances)
ALICE_DEFAULT_MODELDefault model, e.g. openrouter@anthropic/claude-opus-4.6
ALICE_USER_IDUser ID (single-user mode)
ALICE_AUTH_SECRETLogin authentication secret
ALICE_HOSTHost address, e.g. IP:8081
ALICE_BASE_DIRBase directory (auto-set for local install)
ALICE_BIND_ADDRBind address (default: 0.0.0.0, local: 127.0.0.1)
ALICE_PORTListen port (default: 8081)
ALICE_SKIP_AUTHSkip authentication (true for local)
ALICE_SETUP_ENABLEDEnable first-run setup page
ALICE_AUTO_BROWSERAuto-open browser on start
ALICE_PID_FILECustom PID file path
ALICE_SHELL_ENVShell environment description (injected into prompt)
ALICE_INFER_LOG_INLog inference inputs (debug)
ALICE_INFER_LOG_RETENTION_DAYSInference log retention days

Instance Settings (settings.json)

Per-instance config. Values here override environment variables.

KeyTypeDescription
api_keystringLLM API key (overrides env)
modelstringModel identifier (see format below)
user_idstringOwner user ID
privilegedboolEnable script execution & file access
namestringAgent display name
avatarstringAvatar emoji
colorstringTheme color, e.g. #5B5EA6
max_beatsnumberMax heartbeats per inference
history_kbnumberHistory compression target size (KB)
session_blocks_limitnumberMax session blocks (recent memory)
session_block_kbnumberMax size per session block (KB)
safety_max_consecutive_beatsnumberSafety valve: max consecutive beats
safety_cooldown_secsnumberSafety valve: cooldown seconds
billing_urlstringBilling API URL
billing_api_keystringBilling API key

HTTP API Reference

All /api/* routes (except auth) require authentication via alice_session cookie.

Authentication

MethodEndpointDescription
POST/api/authLogin — body: {"secret":"..."}, sets HttpOnly cookie
GET/api/auth/checkCheck if authenticated
POST/api/logoutClear session cookie

Instance Management

MethodEndpointDescription
GET/api/instancesList all instances (with status, knowledge size, etc.)
POST/api/instancesCreate a new instance
DELETE/api/instances/{name}Delete an instance
POST/api/instances/{name}/settingsUpdate instance settings (partial JSON merge)

Messaging

MethodEndpointDescription
POST/api/instances/{name}/messagesSend message to agent — body: {"content":"..."}
GET/api/instances/{name}/replies?after_id={id}Poll agent replies (long-polling with after_id cursor)
GET/api/instances/{name}/history?before_id={id}&limit={n}Load older messages (pagination)

Observation & Control

MethodEndpointDescription
GET/api/instances/{name}/observeAgent status: engine state, inference output, current action
GET/api/instances/{name}/inferenceInference status and streaming output
GET/api/instances/{name}/actionsList recent actions
POST/api/instances/{name}/interruptInterrupt agent (cancel current inference)

Files & Knowledge

MethodEndpointDescription
GET/api/instances/{name}/files/listList workspace files
GET/api/instances/{name}/files/read?path={path}Read file content from workspace
GET/api/instances/{name}/knowledgeGet instance knowledge (knowledge.md)

Fission (Instance Split)

MethodEndpointDescription
POST/api/fission/suggest/{name}LLM-powered knowledge split suggestion (SSE stream)
POST/api/fission/generate-wordsGenerate fission name and letters
POST/api/fission/executeExecute fission: create child instance

Static Files & Proxy

MethodEndpointDescription
GET/serve/{name}/{path}Serve workspace file (requires auth)
GET/public/{name}/{path}Serve public file from apps/ dir (no auth)
ANY/proxy/{port}/{path}Reverse proxy to localhost:{port} (ports 1024-65535)

Setup & Misc

MethodEndpointDescription
POST/api/setupFirst-run setup (create instance + configure)
POST/api/frontend-logFrontend error logging
